## Reproducing test steps

Run on Ubuntu 22.04

```bash
python3 -V # Python 3.8.13

 cat <<EOF >>./pyproject.toml
[semantic_release]
commit_message = "{version}\n\nAutomatically generated by python-semantic-release"

[semantic_release.branches.main]
match = "(main|master)"

prerelease = false

[semantic_release.branches.dev]
match = "dev-.*"
prerelease_token="dev"
prerelease=true

[semantic_release.remote.token]
env = "GH_TOKEN"
EOF

```
